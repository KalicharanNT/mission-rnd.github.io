Getting Started

 

Each project has a set of project pages that will walk you through the concepts you need to learn for that project. Project pages have the following structure:

  • Project Overview
  • Concepts
  • Setup Instructions
  • Follow-Along Tutorial
  • Brief description of the code activities in the project.

Each project is broken down into a set of activities.  Once you code an activity, you can verify your code by running a set of tests for that activity that are provided with the project. This project content is available in a public github repo. You will use the project advisor tool to setup the project on your laptop, run the tests after coding an activity and push your changes to a personal github repo for future reference.

Here are the steps to get started on the code projects:

  1. Download the project advisor tool - Unzip the file to a local drive (G:\ProjectAdvisor for example).
  2. Start with the JavaScript Intro project and progressively go through the remaining projects.
  3. Each project is broken down into a set of activities. Once you code an activity, you can verify your code by running a set of tests for that activity that are provided with the project.
  4. Follow the instructions on how to run the tool and validate your code changes. The project is considered complete once all the activities of that project are complete.
  5. Information about the projects, project concepts/lessons and Stackoverflow forum links are given below.
