C-LinkedLists

Overview

Linked List problems.

Project Activities
 

numberToLinkedList

Given a number convert that to single linked list (each digit as a node).

012sllSort

Given a single linked list of 0s 1s and 2s ,Sort the SLL such that zeroes will be followed by ones and then twos.

twodigitLinkedList

Given a single linked list (two digits as a node).Convert that to number.

removeEveryKthNode

Given a single linked list, remove every Kth node in the linked list.

sortLinkedList

Given an unsorted single linked list, sort the linked list without using extra array or linked list.
