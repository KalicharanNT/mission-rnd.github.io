Intro to C problems.

Project Activities
 

sum

Write a function which takes two numbers as input and returns sum of two numbers.

swap

Write a function which takes two numbers as input and swaps those two numbers.

count

Given 2 integers (num1, num2), write a function that returns the count of numbers between the num1..num2 that are divisible by num1.

isOlder

Given date of births of two persons, find who is older.

variableArguments

A school has unknown number of students. Write a function that takes a variable number of arguments representing student marks in english and returns the number of students who scored > 90 marks
